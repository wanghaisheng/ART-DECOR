This folder contains third party tools required by ART-DECOR

eXist-db
Contains an eXist-db installer which has been adapted slightly for use with ART-DECOR.

Orbeon
Contains an Orbeon forms web archive for installation in Tomcat. This war is required by ART-DECOR.

Art-Decor installation.pdf
Contains detailed installation instructions.

Note:
This folder no longer contains the application & data packages. All application packages can now be directly installed from the Nictiz application repository at http://decor.nictiz.nl/apps/public-repo
